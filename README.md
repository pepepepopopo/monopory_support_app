# DigiPoly

モノポリーの資産管理をスマホで行うサポートアプリ。
各プレイヤーが自分の端末から参加し、お金のやり取りをリアルタイムで管理できる。

## 技術スタック

| 区分 | 使用技術 |
|------|----------|
| フロントエンド | Vite + React + TypeScript + TailwindCSS + DaisyUI |
| バックエンド | Ruby on Rails (API mode) + ActionCable |
| データベース | PostgreSQL (Neon) |
| 環境構築 | Docker + docker-compose |
| デプロイ | Render |

## 機能一覧

### フェーズ1: ゲーム設定 (実装済み)

- **ゲーム作成** - ホストがルームを作成し、参加コード・QRコードを発行
- **ゲーム参加** - 参加コードまたはQRコードでゲームに参加
- **プレイヤー設定** - 名前・識別カラーを登録
- **初期所持金設定** - ホストが開始前に初期金額を設定
- **リアルタイム同期** - ActionCableでプレイヤーの参加・退出をリアルタイム反映
- **ゲーム開始** - ホストが開始ボタンでプレイ画面に全員遷移

### フェーズ2: 送金管理 (実装済み)

- **送金** - プレイヤー間の送金（1対多に対応）
- **銀行機能** - ホストが銀行として入出金を操作
- **クイック入力** - よく使う金額のボタンで素早く入力
- **残高表示** - 全プレイヤーの所持金をリアルタイム表示
- **取引履歴** - 送金ログを時系列で確認
- **ゲーム終了** - ホストがゲームを終了し、最終順位を表示

### フェーズ3: ルーレット機能 (構想段階)

- サイコロ・イベント抽選をアプリで代替
- アニメーション表示
- 結果の全体共有

## セットアップ

### バックエンド

```bash
cd back
docker-compose up
```

### フロントエンド

```bash
cd front
npm install
npm run dev
```

## 画面遷移

```
トップ (/)
  └── ゲーム作成 (/games)
        ├── 参加画面 (/games/:token/join)
        ├── 設定画面 (/games/:token/startSetting)
        ├── プレイ画面 (/games/:token/play)
        └── 結果画面 (/games/:token/result)
```
