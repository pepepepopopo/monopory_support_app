# モノポリーの物件組み合わせ最強は？色別の強さと集め方を徹底解説

モノポリーで勝つには「どの色の物件を集めるか」が最も重要です。でも、なぜオレンジが強いのか？ダークブルーは本当にお得なのか？感覚ではなく**数字で**物件の強さを比較してみましょう。

この記事では、物件価格・建設費・家賃・止まる確率をもとに、**各カラーグループの本当の実力**を解説します。

## モノポリーの物件を評価する3つの基準

物件の強さは以下の3つで決まります。

- **投資回収率** — 物件+投資費用に対して、家賃でどのくらい早く元が取れるか
- **止まる確率** — そもそもどのくらいの頻度で相手が止まってくれるか
- **必要資金** — 独占+建設にいくらかかるか。手持ちで現実的に達成できるか

高い家賃でも誰も止まらなければ意味がないし、安く建てられても家賃が低すぎれば勝ちきれません。この3つのバランスを見ることが大事です。

基本的には小さな投資で収入が得られる場所と投資額は必要だが収入が多い場所の二つを獲得することがセオリーです！

例：鉄道（空港）　＋　黄色
　　ライトブルー　＋　黄色

| 項目 | データ |
|---|---|
| 物件数 | 2 |
| 物件価格合計 | $120（$60+$60） |
| 建設費 | $50/軒 |
| 家3軒ずつの総投資 | $420 |
| 家3軒の家賃 | $180 / $180 |
| 独占時の家賃（家なし） | $4 / $8 |

**評価**: 投資額が最も安いのが唯一の利点。しかし家賃が低すぎて勝ちきれない。ボードの序盤に位置するため、GO直後は通過されやすい。すぐに家を建てれるため家の数を減らし、他の人が建てれないようにする動きができる。序盤に手に入れ、ホテルにできるとましな収入となるのでとりあえず家を建てるという戦略も取れる**序盤に偶然揃ったら建てる程度**。

鉄道・空港で安定した収益を得つつ家を建てるという戦略は最初に破産することがほぼないと言えるほど強い

## 実戦での物件の集め方

### 序盤（1〜3周目）：とにかく買う

- 止まった物件は**基本的に全部買う**
- この段階では「どの色を集めるか」より「選択肢を増やす」ことが重要
- オレンジ・ライトブルーに止まったら最優先で購入

### 中盤（独占が見えてきたら）：交渉で揃える

- 自分が2つ持っているグループの残り1つを交渉で手に入れる
- 交渉のコツ: 相手にも独占の材料を渡す「Win-Winトレード」
- 例: 「私のレッド1枚とあなたのオレンジ1枚を交換＋$100」

### 終盤（独占後）：家を建てる

- まず家3軒を目指す（コスパ最適ライン）
- 余裕があれば家4軒で止めて「家の在庫を枯渇させる」上級テクニック
- ホテルに進化させると家が市場に戻るため、他プレイヤーが建設可能になるリスクがある

## なぜ「オレンジ最強」なのか：刑務所との関係

オレンジが最強の最大の理由は**刑務所の存在**です。

- モノポリーでは「刑務所へ行け」マス、チャンスカード、ゾロ目3連続など、刑務所に入る機会が多い
- 刑務所から出るとき、サイコロの出目6〜8（最も出やすい範囲）でオレンジに着地する
- つまり、**ゲームが進むほどオレンジに止まる確率が上がる**
- 統計的に、オレンジのIllinois Aveはボード全マスの中で**最も止まりやすい物件の1つ**

この「刑務所→オレンジ」の導線を理解しているかどうかで、モノポリーの勝率は大きく変わります。
