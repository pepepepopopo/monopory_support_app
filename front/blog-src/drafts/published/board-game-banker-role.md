# ボードゲームの銀行役とは？モノポリー・人生ゲームで銀行係がやること完全まとめ

「誰が銀行役やる？」

ボードゲームを始めるとき、この一言で沈黙が流れることがあります。銀行役は大変なイメージがあるからです。

でも実際、銀行役がどんな仕事をするのか、具体的に説明できますか？「なんとなく大変そうだから避けたい」という漠然とした印象を持っている方が多いかもしれません。一方で、銀行役を引き受けたものの、何をどうすればよいかわからず焦った経験がある方もいるでしょう。

この記事では、モノポリーと人生ゲームそれぞれの銀行役がやることを具体的にまとめ、なぜ大変なのか、そしてどうすれば楽になるかを解説します。初めて銀行役をやる方にも、久しぶりにやる方にも参考になる内容です。

## ボードゲームの銀行役とは

銀行役（バンカー）とは、ゲーム中に**銀行の代理として現金・物件・各種カードを管理・払い出しするプレイヤー**のことです。

なぜ銀行役が必要かというと、ボードゲームの進行には「誰かが全員のお金の出入りを管理する」機能が必要だからです。コンピューターゲームなら自動で処理されますが、アナログゲームではプレイヤーの一人がその役割を担います。

銀行役は**プレイヤーとしても参加しながら**、同時に銀行業務も行います。これが「大変」と感じられる主な理由です。

なお、「銀行役＝お金を管理する人」というシンプルな役割のように見えますが、実際には「お金の払い出し」「物件の管理」「競売の仕切り」「破産処理」など、ゲームの進行を支える多くの業務が含まれます。ゲームによっても業務の内容が大きく異なるため、ゲームごとのルールを把握しておくことが重要です。

銀行役を「誰かが犠牲にならないといけない役割」と捉えるのではなく、「アプリと分担できる役割」として捉え直すことで、ゲーム全体の体験が大きく変わります。

## モノポリーの銀行役がやること

モノポリーの銀行役は、以下の業務を担当します。

### ゲーム開始前

- 各プレイヤーに初期資金$1,500を配布（版によって紙幣の内訳は異なる）
- 残りの紙幣を銀行として管理
- 物件カード（22枚）、家（32軒）、ホテル（12個）を手元に整理

配布にかかる時間を短縮するために、あらかじめ「スターターセット」として紙幣をまとめておくのがコツです。4人分ならゲーム前に4つの束を作っておけば、配布がスムーズになります。

### ゲーム中の主な業務

| タイミング | 銀行役の業務 |
|---|---|
| GOを通過したとき | 通過したプレイヤーに$200を支払う |
| 物件を購入するとき | 物件カードを渡し、購入代金を受け取る |
| 家・ホテルを建設するとき | 家/ホテルを渡し、建設費用を受け取る |
| 家・ホテルを売却するとき | 家/ホテルを受け取り、半額を支払う |
| 税金・罰金カードが出たとき | 支払われたお金を銀行として受け取る |
| 抵当に入れるとき | 物件カードを預かり、抵当額を支払う |
| 抵当を解除するとき | 物件カードを返し、抵当額+10%を受け取る |
| 競売が行われるとき | 競売を仕切り、落札代金を受け取る |
| プレイヤーが破産したとき | 物件の競売・資産の整理を取り仕切る |

ゲームが進むほど処理量は増えます。特に後半は家やホテルの建設・売却が頻繁に起き、さらに抵当・破産処理が重なることもあります。参加人数が4〜6人になると、誰かが止まるたびに銀行役が動く状態が続きます。

### ゲーム終了後

- 残った紙幣を回収
- 全物件カードと家・ホテルを回収・整理

これだけの業務をゲームプレイと並行して行う必要があります。

## 人生ゲームの銀行役がやること

人生ゲームの銀行役はモノポリーと比べると、物件管理や競売がない分シンプルですが、独自の複雑さがあります。

### ゲーム開始前

- 各プレイヤーに初期資金を配布（通常$30,000〜$50,000、版によって異なる）
- 職業カード・給与カードをシャッフルして配布準備
- 保険・株券・お宝カード類の整理

### ゲーム中の主な業務

| タイミング | 銀行役の業務 |
|---|---|
| 給与日マスを通過したとき | 各プレイヤーの職業に応じた給与を支払う |
| 税金マスに止まったとき | 税額を受け取る |
| 保険を購入するとき | 保険料を受け取る |
| 保険を使うとき | 保険金を支払う |
| 株を購入するとき | 株券を渡し、購入代金を受け取る |
| 約束手形（借金）を発行するとき | 約束手形カードを渡し、貸付金を支払う |
| 精算時の借金回収 | 約束手形1枚あたり元金×1.25で返済を受け取る |

人生ゲーム特有の難しさは**給与計算**にあります。プレイヤーによって職業が異なるため、「Aさんは医者だから$60,000、Bさんは教師だから$40,000」と毎回確認しながら支払う必要があります。6人プレイなら給与日のたびに6人分の金額を確認・払い出しするため、時間がかかります。

**モノポリーとの主な違い**

| 比較項目 | モノポリー | 人生ゲーム |
|---|---|---|
| 物件管理 | あり（家・ホテルも） | なし |
| 競売業務 | あり | なし |
| 給与の支払い | なし | あり（職業別の計算が必要） |
| 借金の利息計算 | なし | あり（精算時に25%） |
| ゲームの停止頻度 | 高い（毎ターン何らかの処理） | 中程度 |

人生ゲームの銀行役の難しさは、**職業別の給与計算**と**精算時の利息計算**に集中しています。逆に言えば、この2点さえ対処できれば、モノポリーよりも銀行役の負担は少なくなります。

精算時の利息計算は特に忘れがちです。ゲーム中に借りた約束手形（$20,000単位）ごとに25%の利息がかかるため、複数枚借りているプレイヤーがいると、精算だけで数分かかることがあります。事前に「精算は一人ずつ順番にやる」と決めておくと混乱を防げます。

## なぜ銀行役は「大変」なのか？3つの専門的分析

### ① 認知的資源の枯渇（Cognitive Overload）

銀行役は自分の番だけでなく、他プレイヤーのターンでも常に「計算・確認・送金」を行わなければなりません。これにより、本来「どの物件を交渉すべきか」に向けるべき脳のエネルギーが、単純な算術に奪われてしまいます。これはプレイヤーとしての圧倒的なハンデキャップです。

認知心理学でいう「デュアルタスク」の状態——2つの異なる作業を同時にこなすことは、どちらかの精度を必ず下げます。銀行業務に集中するほどゲームの戦略が疎かになり、ゲームの戦略を考えるほど銀行処理が遅くなる、という悪循環が生まれます。

### ② 計算ミスの「責任」と「修復不可能」なトラブル

アナログのやり取りでは、後から「さっきお釣りをもらっていない」「家賃を二重に払った気がする」という疑念が生じた際、検証する方法がありません。銀行役は全責任を負わされる立場にあり、これがプレイヤー間の心理的な摩擦を生む原因となります。

記録がない状態でゲームを進めるほど、こうした疑念が積み重なります。特に長時間ゲームでは、序盤の些細なミスが精算時まで引きずられることがあります。

### ③ ゲームの流れを自分が止めてしまう

物件購入・家の建設・抵当の計算など、銀行役が処理している間、他のプレイヤーはゲームが進むのを待ちます。「早く計算して」というプレッシャーがミスをさらに誘発します。

銀行役のプレイヤーが一人で抱え込むほど、ゲーム全体のテンポが銀行役の処理速度に依存します。これは特に銀行役に不慣れなプレイヤーがその役割を担ったときに顕著です。「ゲームが止まっている時間」は全員のストレスになり、銀行役への不満にもつながります。

## 銀行役をスムーズに進める3つの実践的なコツ

「大変」とわかっていても、銀行役を引き受けることになった場合のために、実践的なコツを紹介します。

### コツ1：紙幣を種類別に整理して手元に置く

$1、$5、$10……と種類別に積み上げておくことで、払い出しの速度が大幅に上がります。「$300の支払いなので$100×3枚」と素早く取り出せる状態を常に保ちましょう。ゲーム中盤以降は小額紙幣が不足しやすいため、高額紙幣を両替しておく余裕があるうちに行うのがポイントです。

### コツ2：計算は声に出して確認する

「$400の家賃、$100×4枚で払います」「抵当解除は$30に10%の$3を足して$33ですね」と声に出すことで、二重チェックになります。全員が聞いている状態で処理することで、「計算が違う」という後からの指摘を防げます。

### コツ3：怪しい取引はその場で全員に確認する

「さっきの家賃、$200でよかったですか？」と、その場で全員に確認する習慣をつけましょう。後から「払いすぎた」「もらっていない」という主張が出ても、その場で全員が確認済みという共通認識が生まれます。アプリで送金記録を残しておけばさらに確実です。

## 銀行役は誰に向いているか

「計算が得意な人」が銀行役に選ばれがちですが、実は**ルールを正確に把握している人**の方が向いています。計算量そのものは電卓やアプリで補えますが、「この状況ではどのルールが適用されるか」の判断はルール理解が必要だからです。

初めてモノポリーや人生ゲームをプレイする人が銀行役をやると、ルールを確認しながら計算も行うという二重の負担が生まれます。ゲームの楽しさを覚える前に疲弊してしまうこともあります。可能であれば、経験者が銀行役を担うのが理想的です。

ただし、アプリで送金処理・残高管理を行えば、「計算が不安」という問題は大幅に解消されます。初心者でも安心して銀行役を担える環境を作ることで、毎回同じ人が銀行役を押し付けられる状況を変えられます。

## よくある疑問：銀行役に関するQ&A

**Q. 銀行役のプレイヤーは不利になる？**

A. アナログでやる限り、不利になりやすいです。計算と確認に脳のリソースを取られるため、自分の戦略を考える余裕が減ります。長時間ゲームになるほどこの差は大きくなります。アプリで銀行処理を補助すれば、この差はほぼなくなります。

**Q. 途中で銀行役を交代してもいい？**

A. ルール上の制限はありません。長時間ゲームでは、疲れてきたタイミングで交代するのも選択肢です。ただし途中交代では「どこまで処理したか」の引き継ぎが必要なため、取引履歴の記録があると安心です。アプリを使っていれば、交代後も同じ履歴を全員で確認できます。

**Q. 銀行役は何人でやってもいい？**

A. ルール上は1人が担当するのが基本ですが、2人で分担（一方が現金担当、一方が物件担当）しても問題ありません。参加人数が多いときや、全員が初心者のときは分担するのも一つの方法です。アプリで残高管理を行えば、現金の受け渡しのみに集中できるため、銀行役1人でも負担が大幅に減ります。

## まとめ：銀行役を「仕事」から「ゲーム体験の一部」へ

ボードゲームの銀行役は、本来ゲームの経済システムを支える楽しい役割であるべきです。しかし、アナログの限界がそれを「苦行」に変えてしまっています。道具を使えば変えられます。

モノポリーの銀行役は物件管理・競売・破産処理と業務が多岐にわたり、人生ゲームの銀行役は給与計算・利息処理という独自の複雑さを持ちます。どちらも「計算係」として孤軍奮闘するより、アプリと役割分担する方が全員にとって快適です。

送金記録・残高管理といった煩雑な作業をアプリに任せ、人間は「交渉」「判断」「勝負」というボードゲームの本質に集中する——これが現代のボードゲームを快適に楽しむ形です。

銀行役の負担をゼロにし、全員が対等に楽しめるゲーム環境を整えること。それが、あなたのボードゲームナイトを何度でも成功させるための「最高の一手」です。
